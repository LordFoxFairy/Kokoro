给我的小满咖啡馆出一套小红书开业营销物料，下周六开业。

好。一套五件，全用同一个暖色调，发出去像一家店的——

  1. 主视觉海报（开业那张，能直接当封面）
  2. 标题 + 正文文案（小红书笔记照着发）
  3. 一句 slogan（贴杯套、写黑板都行）
  4. 朋友圈九宫格（开业当天发）
  5. 配色 + 话题标签（之后follow发都对得上）
在用 · 营销 Agent · 一套 5 件

右边这一套都是"清新米"主题，颜色是统一的。主视觉那张我放大了，其它几件配着它走。要换风格我整套一起刷。

文案再口语一点，别太广告腔。"小满"是节气，想留着这个意思。

改了。正文换成像跟朋友说话那样，slogan 顺着"小满"——刚刚好、不贪多的意思。整套的标题、九宫格配字我都跟着同步过了，不会一件一个调。
