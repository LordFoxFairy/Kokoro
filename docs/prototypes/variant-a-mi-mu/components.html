基础原子 · 补缺

v0.3 第三轮补：头像 / 角标 / inline link / kbd / 危险按钮 / loading 按钮 / disabled 输入
.avatar · .avatar--sm/md/lg/xl

头像 · 四档尺寸

小 心 秋 林

默认暖底 + 木色字。需要图片时用 <img> 占位即可。

.badge · .badge--dot / --soft / --danger

小角标

3 12 ! ✓
小

用在 nav-item / avatar / chip 上，永远是补充信息，不要替代文字。

.inline-link · 行内链接

行内链接

想接着做的话，看看 Pro。 上次给你做的那份生日卡也是同一套版式。

下划线是暖色软线 + hover 加深，不用蓝色链接。

.kbd · 键盘标记

键盘标记

⌘K ⏎ Esc ⌘+S

底边加粗 1px 做按键凸感。配 .kbd-hint 写说明文字。

.btn--danger

危险按钮 · 暖红

用暖偏红（不是冷红），配合二次确认 modal（见 circuit-breakers）。

.btn.is-loading

Loading 按钮 · 暖色细圈

字隐 + 暖色细圈呼吸。按钮宽度不变，不抖。

.input-pill.is-disabled

输入禁用态 · 额度用完时

底色变暗一档，整体降明度。配 .limit-notice 一起用，告诉用户"不是坏了，是该停了"。

.save-state

自动保存指示

空闲 已保存 · 刚刚
保存中 正在保存…

canvas 顶栏右侧，绿点 = 安静 / 木色呼吸点 = 在写。

.message--error

错误态消息 · 心人格

这一步我没成
Canvas 那边没接上。你和我说过的、我们刚聊出来的两个想法，我都先存下来了——不会因为这个错误丢掉。

暖偏红软底 + 左 2px 强红线；区别于正常 ai 文本但不刺眼。Copy 必须以"我"开头承担责任。

.limit-notice

额度到达 inline 提示

今天的额度差不多了。 要不要先暂停一下？想接着做的话，看看 Pro。

放在 composer-dock 内、input 之上。暖警告色，不油腻，"看看 Pro"是 inline-link 不是 cta。
