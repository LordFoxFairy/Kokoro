# ADR-005 · 商业模式：freemium + 创作者 Pro

- **日期**：2026-05-21
- **状态**：accepted（起点结构，具体定价待真实数据校准）
- **决策者**：Claude（用户全权授权代决）
- **关联**：[ADR-001](./ADR-001-product-personality.md)、[ADR-002](./ADR-002-target-users.md)、[ADR-004](./ADR-004-north-star-metric.md)、[01-strategy/business-model.md](../product/01-strategy/business-model.md)

---

## 决策

**起点 = freemium + 单档 Pro 订阅。** 长期叠加团队版（v2+ 评估）。

### 免费层（核心免费）

- 每天 N 次 Canvas 生成（具体额度待运营拍）
- **产物自带 Kokoro 印记，自然档不可关闭**
- 标准模板访问
- **分享 link / OG / 公开页（增长一等公民，不上付费墙）**
- 社区模板提交

### Pro（¥40-80 / 月，价格区间待 AB 测试）

- 无限 / 高额度 Canvas 生成
- **印记可弱化到「自然-」档位，但不可完全去除**（红线）
- 高级 / 早访模板
- 私有保存（产物不进社区）
- 优先模型 / 更快响应

### v2+ 评估
- 团队版（¥99-299 / 座 / 月）
- B 端 / 企业定制（独立评估，不在本 ADR）

---

## 理由

1. **与目标用户契合**：T0（创作者 + 创业者）对 SaaS 订阅有付费意愿但价格敏感
2. **与增长引擎兼容**：免费层覆盖分享场景（[ADR-004](./ADR-004-north-star-metric.md) 的 WAS 需要免费用户撑量）
3. **与心人格不冲突**：避免广告 / 弹窗 / 强行升级提示
4. **未来扩展空间**：团队版是 v2+ 增长上限

---

## 否决项

| 候选 | 否决理由 |
|---|---|
| **单次付费**（每份产物付钱） | 与"分享一等公民"对立 |
| **广告变现** | 与心人格根本冲突 |
| **Pro 完全去除印记** | 破坏增长引擎（产物即广告失效） |
| **早期上付费墙** | 引擎转不起来 |
| **个性化数据卖给三方** | 与[09-safety/privacy.md](../product/09-safety/privacy.md) 信任契约冲突 |

---

## Tradeoff

- **免费层成本压力大** → 需要早期资本预备 / 模型成本谈判 / 速率限制控量
- **Pro 单价相对低** → 规模化时间长，初期 ARR 不会爆发
- **"印记可弱化但不可去"会有部分高级用户抱怨** → 接受，这是增长护城河
- **额度限制 vs 流畅体验的取舍** → 必须做"额度透明显示"，不要静默卡死

## 后果

- [voice-and-tone.md](../product/02-personality/voice-and-tone.md) 升级 Pro 引导文案：不油腻、不催促（"额度用完了。要不要先暂停一下？"）

- 速率限制 / 额度透明显示 应作为 MVP 的一部分，非后置
