# 增长指标

> 🟢 **北极星已锁定**（[ADR-004](../../decisions/ADR-004-north-star-metric.md)）：**WAS = 周活跃产物分享数**。
> 必须配反作弊规则（同 IP / 同设备 / 短时窗去重）。
> 一张表答完三件事：北极星是什么、它由什么驱动、什么时候要拉警报。目标值仍标「仅参考」——等真数据覆盖。

---

## 1. 北极星候选 ⭐

### 选项对比

| 候选 | 长这样 | 优 | 劣 |
|---|---|---|---|
| **A. 周活跃产物分享数（WAS）** | "本周被分享出去的产物总数" | 直接反映"产物即广告"循环健康度；可拆为人均 / 总量 | 易被刷（用户内部互分享） |
| **B. 周新增可复用模板数（WNT）** | "本周新增并被至少 3 次复用的模板数量" | 反映模板沉淀循环；硬指标 | 早期数太小；滞后（要等"被复用 3 次"） |
| **C. 周活产物完成数（WAP）** | "本周完成的产物总数" | 朴素直接、产品价值底座 | 不反映传播 |
| **D. 月活创作者数（MAC）** | "月内完成 ≥ 1 个产物的用户" | 行业标准、好对比 | 太通用，不体现 Kokoro 差异化 |

### Claude 推荐：A（WAS）

理由：
- Kokoro 的核心命题是「产物即广告」——北极星必须直接刻画这个命题
- WAS 同时压住了产物质量（不好用没人完成）、产品价值（产物对用户有意义才会做）、传播力（分享触发率）三件事
- B 滞后、C/D 不差异化

**但**：WAS 必须配合反作弊（同 IP / 同设备 / 短时间内重复分享同一产物视为一次）。

### 季度建议目标（仅参考）

| 季度 | 目标 |
|---|---|
| Q1（产品上线） | WAS = 500 / 周 |
| Q2 | WAS = 5,000 / 周 |
| Q3 | WAS = 30,000 / 周 |
| Q4 | WAS = 100,000 / 周 |

数字粗糙，看实际曲线再校准。

## 2. 输入指标（北极星的驱动因子）

### 用户量类

| 指标 | 定义 | 草稿目标（仅参考） |
|---|---|---|
| **DAU** | 日内活跃用户（打开 + 至少一次有意义交互） | 上线 6 个月达 1万 |
| **新用户产物完成率** | 注册后 7 日内完成第一个产物的比例 | > 50% |
| **新用户首产物耗时** | 注册到完成第一个产物的中位数 | < 15 分钟 |

### 分享类（① 循环）

| 指标 | 定义 | 草稿目标 |
|---|---|---|
| **分享触发率** | 完成的产物中触发了"分享"动作的比例 | > 35% |
| **分享链接点开率** | 公开 share link 的 UV / 分享次数 | > 4 人/分享 |
| **OG 图点击率** | OG 图被动曝光 → 点入落地页 | > 2% |
| **分享落地→新用户率** | share link 落地 UV 中转化为新注册的比例 | > 4% |
| **印记保留率** | 分享产物中保留默认或更强印记档位的比例 | > 80% |

### 模板类（② 循环）

| 指标 | 定义 | 草稿目标 |
|---|---|---|
| **产物→模板转化率** | 完成产物中被另存为模板的比例 | > 3% |
| **模板复用率** | 模板被使用次数 / 模板存量 | > 平均 5 次/模板/月 |
| **模板使用→完成率** | 用模板开始的产物的完成率 | > 60% |
| **新增可复用模板数** | 本周新模板中至少被复用 3 次的数量 | 见北极星候选 B |

### K-factor 类

| 指标 | 定义 | 草稿目标 |
|---|---|---|
| **总 K** | 见 [viral-loops.md](./viral-loops.md) | 6 个月内 > 1.0 |
| **T1（share link）贡献** | i₁·c₁ | > 0.3 |
| **T2（OG 图）贡献** | i₂·c₂ | > 0.3 |
| **T3（模板）贡献** | i₃·c₃ | > 0.1 |
| **T4（邀请）贡献** | i₄·c₄ | > 0.05 |
| **cycle_time** | 一轮传播周期 | < 5 天 |

## 3. 健康指标（看根，不看叶）

### 留存

| 指标 | 定义 | 草稿目标 |
|---|---|---|
| **D1 留存** | 注册次日访问比例 | > 35% |
| **D7 留存** | 注册第 7 日访问比例 | > 20% |
| **D30 留存** | 注册第 30 日访问比例 | > 12% |
| **WAU/MAU** | 周活/月活比 | > 0.4（粘性） |

### 产物质量

| 指标 | 定义 | 草稿目标 |
|---|---|---|
| **产物完成率** | 开始的产物中完成的比例（区分 abandon vs export） | > 55% |
| **产物再编辑率** | 产物完成后 7 日内被重新编辑的比例 | > 25% |
| **产物 NPS** | 完成时弹一下"你满意吗"（克制使用） | > 7/10 |

### 气质 / 印记

| 指标 | 定义 | 草稿目标 |
|---|---|---|
| **印记保留率** | 分享产物中保留 ≥ 默认档位印记 | > 80% |
| **印记自愿升档率** | 用户主动选「醒目」档位的比例 | > 8% |
| **印记降档放弃率** | 想降到「隐身」但因 Pro 门槛放弃的比例 | 监控（无目标） |

## 4. 警戒指标（亮红灯就要查）

### 滥用

| 指标 | 阈值（亮红灯） | 处理 |
|---|---|---|
| **分享但被举报率** | > 1% | 自动审查 + 人工抽检 |
| **模板被删比例** | > 5% | 自动过滤规则收紧；审核标准复盘 |
| **垃圾账号占比** | > 2%（注册后 24h 无任何产物） | 反欺诈规则更新 |
| **NSFW 漏检率** | 任何漏检都要复盘 | 升级模型护栏 |

### 漏斗劣化

| 指标 | 阈值 | 含义 |
|---|---|---|
| **分享触发率连续 4 周下降** | 任意 | 完成体验出问题 / 分享按钮被埋深 |
| **印记保留率 < 70%** | 任意 | 默认档位太强 / 印记被嫌弃 |
| **新用户产物完成率 < 30%** | 任意 | 首屏 chip 选错 / AI 生成质量降 |
| **K 连续 2 月下降** | 任意 | 核心循环失效 |

### 商业风险

| 指标 | 阈值 | 含义 |
|---|---|---|
| **Pro 用户流失率** | > 10%/月 | Pro 价值感不够 |
| **每用户模型成本** | > 单用户预期 LTV / 36 月 | 烧不起，必须限额或涨价 |

## 5. 看板分层

为了避免每天 50 个指标盯不过来：

| 看板 | 频率 | 受众 | 包含 |
|---|---|---|---|
| **北极星看板** | 周 | 创始团队 | WAS + 4 个 K 分项 |
| **健康看板** | 周 | 产品团队 | 留存 + 完成率 + 印记保留 |
| **警戒看板** | 日 | 全员 | 警戒指标，越线弹通知 |
| **细节看板** | 按需 | 数据 / 产品负责人 | 全部输入指标 |

## 7. 数据决策原则

- **指标矛盾时**：北极星 > 健康 > 输入。WAS 上涨但留存崩 → 先救留存。
- **目标值仅参考**：所有"草稿目标"在拿到 4 周真实数据后必须重新校准。
- **不为指标而指标**：警戒指标更重要，因为它们反映"产品健康"而不是"业绩"。
- **指标不解释产品**：产品决策靠定性 + 用户研究 + 直觉。指标只用于**否决**和**校验**。
