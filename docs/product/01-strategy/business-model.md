# 商业模式

> 🟢 **已锁定**（[ADR-005](../../decisions/ADR-005-business-model.md)）：freemium + 创作者 Pro。
> 免费层覆盖核心 + 分享。Pro 解锁额度 + 弱化印记（不可去除）+ 高级模板。

## 约束

- 增长玩法靠**分享 + 模板**，意味着**早期需要免费层够好用**才能撑住分享
- "心"人格 + 视觉产物 = 创作者人群，**他们对 SaaS 订阅有一定付费意愿**但价格敏感
- 不能让"商业化"破坏气质（避免广告 / 弹窗 / 强行升级提示）

## 三档候选模式

### A. 经典 freemium + 创作者订阅
- 免费：每天 N 次 Canvas 生成，水印产物，标准模板
- Pro（¥40-80/月）：无限生成，去水印（但保留品牌底色）、高级模板、私有保存
- 推荐人群：内容创作者 / 创业者

### B. 免费 + 出品价（每份产物付钱）
- 免费：随便试，导出限制
- 单次：¥3-5 / 份高分辨率导出
- 推荐人群：低频但要质量的用户
- 缺点：分享一等公民跟"导出付费"打架

### C. 免费 + 团队版
- 个人完全免费（带气质印记 + 分享）
- 团队版（¥99-299/座/月）：协作 / 共享模板 / 品牌定制
- 推荐人群：早期增长靠个人，后期靠团队

## 我的初步推荐

**起点用 A，长期叠加 C。** 理由：
- A 与增长玩法兼容（免费层覆盖分享场景）
- C 是未来 B 端故事的起点
- B 跟"分享一等公民"对立，否决

## 必须避免的反例

- ❌ 早期就上付费墙 — 引擎转不起来
- ❌ 广告变现 — 跟"心"人格全冲突
- ❌ 个性化数据卖给三方 — 违反"心"信任契约
- ❌ 让 Pro 用户能"完全去除 Kokoro 印记" — 引擎失效

## 待答题
