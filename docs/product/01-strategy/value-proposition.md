# 价值主张

> 待 [target-users.md](./target-users.md) 锁定后，本文件应针对 T0 用户写出最终版价值主张。
> 当前为通用版草稿。

## 6 部分 JTBD 模板

| 部分 | 内容（草稿） |
|---|---|
| **WHO** | 想把脑里想法变成可分享视觉产物的内容创作者 / 新手创业者 |
| **WHY** | 现有工具（PPT / Figma / Canva / ChatGPT）要么门槛高，要么没人格，做出来的东西不能"携 Kokoro 气质"传播 |
| **BEFORE**（旧方式） | 在多个工具间切换 → 自己排版 → 输出 → 截图 / 导出 → 配文发布。AI 工具能写但不能做，设计工具能做但不能想 |
| **HOW**（Kokoro 怎么做） | 一句话告诉 Kokoro 你想做什么 → Kokoro 在 Canvas 里直接做出来 → 一键分享链接 / 转模板 |
| **AFTER**（新结果） | 产物自带 Kokoro 气质印记 → 朋友看到一眼识别 → 朋友也来用 → 用户成为传播节点 |
| **ALTERNATIVES** | ChatGPT Canvas（无气质）、Gemini（冷蓝、Google 范）、Manus（中性专业）、PPT/Canva（不智能）、v0/Lovable（只 for 开发） |

## 一句话版本（拍朋友圈用）

> "对 Kokoro 说一句话，它帮你做出一份带温度的东西，发给谁谁都问你怎么做的。"

## 三句话版本（落地页用）

> Kokoro 是有人格的通用 AI agent。
> 主战场不是聊天，是把你想做的事变成可分享的视觉产物。
> 每一份产物都自带温柔印记——这是 Kokoro，不是另一个 ChatGPT 截图。
