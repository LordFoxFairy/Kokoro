# 目标用户

> 🟢 **已锁定**（[ADR-002](../../decisions/ADR-002-target-users.md)）：T0 = 内容创作者 + 新手创业者；T1 = 学生 / 设计学生；T2 = 营销 / PM / 教育。
> 推翻需新开 ADR-002a supersede。

## 约束（来自已定决策）

- 「心」人格已锁定 → 目标用户应在"吃温柔治愈调性"的人群里
- 通用 agent + 视觉产出 → 目标用户应"有产物诉求 + 愿意分享"
- 中文 / 东亚为主战场（来自 ADR-001 tradeoff）

## 候选人群（按 fit 排）

| 候选 | 适配度 | 分享倾向 | 视觉诉求 | 早期口碑能力 | 备注 |
|---|---|---|---|---|---|
| **内容创作者**（小红书 / B 站 / 公众号） | 高 | ⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐ | 天然爱分享，自带传播渠道 |
| **学生 / 知识工作者** | 高 | ⭐⭐ | ⭐⭐ | ⭐⭐ | 课件 / 总结 / 笔记拼图需求强 |
| **新手创业者 / Solopreneur** | 高 | ⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐ | 落地页 / 提案 / 营销物料一锅端 |
| **设计师 / 设计学生** | 中 | ⭐⭐⭐ | ⭐⭐⭐ | ⭐⭐⭐ | 但他们专业工具多，门槛挑剔 |
| **营销 / 增长 / 运营** | 中 | ⭐⭐ | ⭐⭐ | ⭐⭐ | 海报 / 投放素材 / 周报 |
| **产品经理** | 中 | ⭐ | ⭐⭐ | ⭐ | PRD / 原型 / 提案 |
| **教育者 / 培训师** | 中 | ⭐⭐ | ⭐⭐⭐ | ⭐ | 课件 / 拼图 / 学习材料 |

## 推荐组合（草稿，待你定）

**T0（首发优先）：内容创作者 + 新手创业者**

- 都吃温柔气质（小红书 / B 站学习区 / 个人成长圈）
- 视觉产出诉求强且**分享是工作的一部分**
- 早期口碑能力强（自带渠道）

**T1（扩散后接入）**：学生、设计学生

**T2（未来）**：营销、PM、教育

## 待答题

- [ ] 海外用户的优先级？（影响是否要英语版 / 国际化策略）
