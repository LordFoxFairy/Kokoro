# 定位

## 三句话定位

> Kokoro 是给"想把脑里的想法快速做成可分享产物"的人用的 AI agent。
> 它在通用 chat 能力之上，主战场是 Canvas 视觉产物。
> 它的灵魂是「心」——柔、温、内观，让每份产出都自带辨识。

## 定位坐标

二维坐标（暂用，待 [01-strategy/differentiation.md](../01-strategy/differentiation.md) 中扩成完整 positioning map）：

```
              通用 / 多用途
                  │
                  │
                  │
 中性专业 ─────────┼───────── 鲜明人格
                  │
                  │
                  │
              垂直 / 单点
```

- **Manus / CoWork / ChatGPT** → 通用 + 中性专业
- **v0 / Lovable / Cursor / Replit** → 垂直 (dev) + 较鲜明人格
- **Pi / Replika** → 垂直 (陪聊) + 极鲜明人格
- **Kokoro 占位**：**通用 + 鲜明人格（柔温内观）**——空白象限

## 与主要参考产品的差异

| 参考产品 | 相似 | Kokoro 不同 |
|---|---|---|
| Manus | 通用 agent + 视觉产出 | Manus 中性专业，Kokoro 有人格 |
| CoWork | Plan-then-Approve + 三栏 | Kokoro 不内嵌 Claude Desktop，web-first |
| Gemini Canvas | chat + Canvas 两栏 | Gemini 冷蓝 / Google 范，Kokoro 暖调 / 心范 |
| ChatGPT | 通用 chat | Kokoro 主打 Canvas 不是 chat |
| Claude Code | 扩展机制（skills/hooks） | Kokoro 借 skills 抽象但非 CLI |

## 不接受的定位

- ❌ "中国版 Manus" — 跟随者叙事，气质死
- ❌ "AI Notion" — Notion 不是 agent，类比错位
- ❌ "ChatGPT killer" — 同质化叙事
- ❌ "agent for everyone" — 口号空泛
