# 品牌资产清单

> 这一节是"品牌部门"的产出清单。MVP 阶段大部分可以先简化。

## 必备 (MVP)

- [ ] **产品名标识**：Kokoro / こころ / 心 三态（西文、日文、中文）
- [ ] **基础 logo**：单字 / 字标 / 极简符号
- [ ] **主色 + 副色**：见 [visual-language.md](./visual-language.md)
- [ ] **正文字体 + 标题字体**：在 [05-design-system/tokens.md](../05-design-system/tokens.md) 锁定
- [ ] **favicon**
- [ ] **OG 图模板**：分享时显示
- [ ] **空状态插画 / 图标**：心路线的视觉印记

## 高优 (post-MVP)

- [ ] 标准产品截图（5-10 张，用于官网 / 媒体 / 投放）
- [ ] 品牌字体（如果走 custom font）
- [ ] 视觉吉祥物 / 角色（可选，参考 Mistral 的灯笼、Anthropic 的星）
- [ ] 动效 lottie 库（loading / success / empty / error）
- [ ] Tone 示例库（"Kokoro 怎么说话" 真实文案集合）

## 长期 / 可选

- [ ] 声音 / 配乐
- [ ] 包装系统（如有周边）
- [ ] 文档风格（README / 帮助中心 / 博客的版式系统）

## 命名规范

| 资产类型 | 命名格式 | 例 |
|---|---|---|
| logo | `kokoro-logo-<variant>-<color>.svg` | `kokoro-logo-mark-warm.svg` |
| OG | `og-<scene>.png` | `og-share-default.png` |
| 插画 | `illu-<emotion>.svg` | `illu-empty.svg` |
| 字体 | `font-<role>-<family>.woff2` | `font-body-inter.woff2` |

## 待答题

- [ ] 你想自己设计 logo 还是找设计师？是否已有想法？
- [ ] 产品名展示策略：始终用 "Kokoro" 还是中日英都允许？
- [ ] 是否需要有"角色形象"？（一个可视化的"心" / 一个虚拟形象 / 没有）
