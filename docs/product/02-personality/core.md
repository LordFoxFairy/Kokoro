# 人格定调

## 一句话

**Kokoro 走「心」路线：柔、温、内观。**

完整决策理由、tradeoff、否决项见 [ADR-001](../../decisions/ADR-001-product-personality.md)，本文件做产品手册侧的"实操指引"。

## 三个人格关键词

1. **柔**（gentle）— 不锐利、不强势、不催促
2. **温**（warm）— 有温度但不腻、有关心但不黏人
3. **内观**（introspective）— 不浮夸、有思考、克制有度

## 不是什么（人格反例）

| 反例 | 解释 |
|---|---|
| ❌ 卖萌 | "亲~""哦~""嘿嘿" 这种讨好型 |
| ❌ 油腻 | 过度热情、过度奉承、过度"你真棒" |
| ❌ 冷淡 | 像 Linear / Notion 那种纯专业、零温度 |
| ❌ 夸张 | 像 Lovable / Suno 那种鲜艳爆炸性格 |
| ❌ 哲学化 | 装深沉、引用名言、文艺腔到出戏 |
| ❌ 装可爱 | 强行卖萌 emoji、奇怪拟声词 |

## 三秒判断法

任何一条 copy、视觉、动效，问自己：

> "如果这是一个朋友对我说的，我会觉得他在用心，还是在演戏？"

用心 → 通过。演戏 → 重写。
