# 模板库（Templates）

> 让用户**不从空白开始**——挑一个半成品，让 Kokoro 把它改成你的。

---

## 一句话用途

把社区里好的产物变成"可一键起飞"的起点，缩短从想法到产物的距离，同时形成创作者飞轮。

## 关键组件

| 组件 | 说明 |
|---|---|
| 分类 chips | 横向滚动，与 canvas-types 对齐 + 兜底"全部" |
| 排序 tab | 热门 / 最新 / 编辑推荐（默认热门） |
| 缩略图网格 | 4 列（桌面）/ 2 列（平板）/ 1 列（手机），圆角 12-16px |
| 模板卡片 | 缩略图 + 名称 + 作者 + 复用数 |
| 详情预览 | 大图左、信息 + CTA 右 |
| 主 CTA | "用这个模板 →" 暖色填充 |
| 类似模板 | 详情页底部一行同类推荐 |
| 搜索 | 顶部一个小入口（可选 MVP） |

## 关键状态

### A. 浏览（默认）

- 分类默认"全部"，排序默认"热门"
- 缩略图懒加载，柔脉动占位
- 滚到底自动加载下一页

### B. 详情

- 进入后大图可点放大、可滚翻多屏（落地页 / 课件常见）
- 右侧 CTA 永远可见（sticky）
- 下方"作者说明"段落，作者写的（限 200 字）

### C. 复用流程（关键转化点）

```
用户点 [用这个模板 →]
  └─ 弹一个轻 modal：
     "想做的是关于什么的？告诉我一句话，我帮你改。"
     [输入框]  [跳过，直接复制]
        └─ 输入 → 进 Canvas，Kokoro 开始按用户场景调整
           或 跳过 → 进 Canvas，原样复制
```

- 复用后 Canvas 顶部保留一行小标记：`基于 @作者 的模板`，可点回原模板
- 用户改完可一键"我也转为模板"，形成飞轮

### D. Empty（分类下无模板）

- 居中插画 + 文案：`这一类暂时还没有模板。要不要做一个？[新建]`
- [新建] 跳 Home 并预选该 chip

### E. Loading

- 网格骨架 + 柔脉动；详情大图骨架

### F. Error（拉取失败）

- 顶部一行小字：`没拿到模板列表。[重试]`
- 已加载的部分保留

### G. First-time（首次访问模板库）

- 顶部一行 onboarding 提示（只此一次）：
  - `模板是别人做好的半成品。挑一个，我帮你改成你的。`
- 关掉后不再出现

### H. 未登录浏览

- 浏览 / 看详情**完全开放**
- 点 [用这个模板] → 轻登录提示 → 登录后无缝继续

## 与增长玩法的关系

- 模板**绝对免费**（破坏增长引擎的不做）
- 复用数公开 → 创作者激励
- 编辑推荐 = Kokoro 团队策展，保证质量底
- 模板首页可在 [home.md](./home.md) 的 chip "更多"里再次出现

详见 [07-growth/templates-flywheel.md](../07-growth/templates-flywheel.md)（如未建则待建）。
