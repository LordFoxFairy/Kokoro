# 我的库（Library）

> 用户在 Kokoro 留下的所有"东西"集中在一处。历史 / 收藏 / 我发布的模板 / 草稿。

---

## 一句话用途

让用户不怕"做完就丢"——所有过去的努力随时找得回。

## 关键组件

| 组件 | 说明 |
|---|---|
| Tab 切换 | 历史 / 收藏 / 我的模板 / 草稿，下划线指示当前 |
| 搜索框 | 在当前 tab 内搜（按名称、内容） |
| 缩略图卡片 | 缩略图 + 名称 + 相对时间。hover 浮出 [⋯] 菜单（重命名 / 删除 / 移到收藏 / 复制链接 / 转模板） |
| 排序 | 默认按"最近活跃"，可切"创建时间" |
| 批量操作 | 长按 / 多选 进入批量模式（删除 / 移动） |
| 空状态 | 每个 tab 独立空状态文案 |

## 四个 Tab

### Tab 1 · 历史（所有对话 + Canvas 项目）

- 默认页
- 包含：所有曾经发起过的对话与 Canvas 项目
- 排序：最近活跃
- 点卡片 → 回到那个对话 / Canvas

### Tab 2 · 收藏

- 用户主动标记 ⭐ 的项目
- 也包含从模板库 / 别人分享页收藏的他人产物
- 显眼区分"我做的"和"我收藏别人的"（小角标）

### Tab 3 · 我的模板

- 用户**主动转为模板**且公开的产物
- 每张卡片显示：复用次数、评论数、最近一次被复用时间
- 提供 [改隐私] [关闭分享] [编辑模板说明] 入口

### Tab 4 · 草稿

- 未完成的对话 / Canvas（用户中途离开或主动暂停的）
- 顶部一行温柔提示：`这些是你没做完的事，要不要捡一个回来？`

## 关键状态

### A. Has-content（默认有内容）

- 缩略图网格，懒加载
- 卡片 hover / 长按出现操作菜单

### B. Empty（每个 tab 独立）

| Tab | Empty 文案 | 行动 |
|---|---|---|
| 历史 | `还没开始过对话。先去说一句？` | [去新对话] |
| 收藏 | `还没收藏任何东西。看到喜欢的点 ⭐。` | — |
| 我的模板 | `还没分享过模板。做了好东西可以转成模板，让别人也能用。` | [去看模板库] |
| 草稿 | `没有未完成的事，干净。` | — |

### C. Loading

- 网格骨架 + 柔脉动，不用 spinner

### D. Search active

- 顶部搜索词高亮
- 无结果：`没找到"XXX"。换个词试试？`

### E. 批量删除确认

- 危险操作，弹 modal（参考 [09-safety/circuit-breakers.md](../09-safety/circuit-breakers.md)）
- copy：`你要删 N 个东西，删了就没了。确认？`
- 按钮：[删了] [算了]

### F. Error

- 顶部一行：`没拿到列表。[重试]`
- 已加载的部分保留

## 与 Sidebar 的关系

- Sidebar 的"最近"是**历史 tab 的子集**（最近 N 个）
- Sidebar 的"📚 库 / 收藏"入口直接进本页
- 本页是 Sidebar 入口的"展开版"
