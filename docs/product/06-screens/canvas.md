# Canvas 视图

> 视觉产物在这里诞生、被看见、被改、被分享。Kokoro 的"引擎"在这里跑。

---

## 一句话用途

让用户**看着东西被做出来**——不是等结果，而是过程本身就是体验。

---

## ASCII 布局速写

```
┌──────┬─────────────────────┬──────────────────────────────────────┐
│ Side │  Chat (左半)         │  Canvas (右半)                        │
│ 240px│                     │  ┌────────────────────────────────┐  │
│      │  ……对话流……          │  │ 项目名 ▾   ↶ ↷   分享   ⋯  ×   │  │
│      │                     │  ├────────────────────────────────┤  │
│      │                     │  │                                │  │
│      │  Kokoro: 做好了，看看？│  │                                │  │
│      │                     │  │       视觉产物 (海报 / 课件)     │  │
│      │  [用户气泡]          │  │                                │  │ ┌─┐
│      │                     │  │                                │  │ │📑│
│      │                     │  │                                │  │ │💬│
│      │                     │  │                                │  │ │🎨│
│      │  ┌──────────┐       │  │                                │  │ └─┘
│      │  │ [+] [↑]   │       │  │                                │  │  (右缘工具条)
│      │  └──────────┘       │  └────────────────────────────────┘  │
└──────┴─────────────────────┴──────────────────────────────────────┘
```

- Canvas 打开后 Sidebar 收窄到 240px（参考 Gemini）
- Chat / Canvas 比例：grid `minmax(360px, 40fr) minmax(420px, 60fr)`；1440 视口下约 chat 464 / canvas 696
- Chat 与 Canvas 之间用 1px 极淡 hairline，不要硬分割线
- Canvas surface 用 `#FCF9F4`（暖中性纸色，与 chat `#FAF7F2` 同色系，避免"两个设计系统"感）
- 顶部工具栏**减负**：左 = 暂时收起 chip + 项目名（含呼吸光标）；右 = 自动保存 micro 文字 + 关闭 ×
  （撤销 / 重做交给 ⌘Z / ⌘⇧Z 键盘快捷键；分享改用 chat 区文档卡片的"分享"按钮，避免重复入口）
- **不要**右缘竖排小图标列（MVP 不需要大纲 / 评论 / 翻译；以后真有需要再做）

## 关键组件

| 组件 | 说明 |
|---|---|
| 顶部工具栏 | **左**：暂时收起 chip + 项目名（hover 可重命名，含呼吸光标暗示"活的"） / **右**：自动保存 micro 文字 + 关闭 ×。撤销 / 重做交给 ⌘Z / ⌘⇧Z；分享改用 chat 区文档卡片入口 |
| 暂时收起 chip | 左上 chip-style 胶囊（圆角 999、左指箭头 + 文字"收起"）。**区别于**右侧 X：X = 关闭并退出 Canvas；收起 = 暂时隐藏 Canvas，回到纯 chat 单栏，下次还能继续 |
| 主内容区 | 视觉产物本体。根据 canvas-type 切换渲染器（海报 / 落地页 / 课件 / 文档 / 想法图） |
| 活的痕迹 | 标题旁呼吸光标 / 正文 `alive-highlight` 软背景 + 手写小注（"Kokoro 刚把这句改轻了一些"）。表达"Canvas 不是死稿" |
| 缩略图浮层 | 多页产物（如课件、落地页多 section）左下角浮一个 mini map |
| 生成指示 | 内容生成中时，主区域顶部一条**淡暖色脉动**进度带（非 spinner） |
| 分享入口 | 已在 chat 区"文档卡片"的"打开 / 分享 / 再调一下"按钮行；Canvas 顶栏不再重复挂分享按钮 |

## 关键状态

### A. 生成中（loading / streaming）

- 主区域顶部一道淡暖色脉动条
- 已生成的部分**就地呈现**（不是等全做完才出）
- Chat 侧同步显示：`正在画……` / `正在写落地页第二屏……`
- 用户可中途打断（输入框输入新指令）

### B. 已完成（has-content）

- 顶部进度带消失
- 顶部工具栏的 **[分享]** 按钮高亮可点
- Chat 侧 Kokoro 说一句：`做好了。看看？`
- 右缘工具条出现"评论"" outline"等次要工具

### C. 编辑中（用户在 Canvas 直接改）

- 进入"就地编辑"：点文字直接改、点元素出现微浮 toolbar
- 顶部多出 [恢复 AI 版] 按钮（可回退到 Kokoro 上一稿）
- Chat 侧出现一行小提示：`你在动画布，我等你弄完再说。`

### D. 错误（生成失败 / 渲染异常）

- 主区域中央：暖色插画式占位 + 一行文案
- copy：`我没把这部分做出来。要不要换种说法再试一次？`
- 顶部 [重试] [关闭] 按钮
- **不展示** stack trace、错误码

### E. 空模板（first-time / 新建未生成）

- 主区域中央一行引导：`告诉我你想要的，左边那栏说一句。`
- 可附一个极淡的画布网格背景，暗示"这里是画的地方"
- 顶部分享按钮**禁用 + 提示** "做出点东西再分享吧。"

### F. First-time（用户第一次看到 Canvas 出现）

- Canvas 从右侧滑入（400-500ms 缓动）
- 第一次会附一个**只出现一次**的轻提示气泡："这里是 Canvas。右边在做的就是你说的事。"
- 用户点 ok 或自动 4s 消失

## 与 Chat 的过渡

| 触发 | 行为 |
|---|---|
| 用户输入框点 🎨 | Chat 收窄到 50% → Canvas 从右侧滑入，初始为"空模板"或"生成中" |
| Kokoro 提议进 Canvas | 对话里出现 [好，做] 按钮，点击同上 |
| 用户点 × 关闭 Canvas | Canvas 滑出，Chat 恢复单栏。**项目不删**，下次在 Sidebar 历史里可找回 |
| Plan mode 中触发 | Plan 通过后自动开 Canvas |

> 切换不要硬切，缓动 400-500ms。Sidebar 同步缩 / 展。
