# 分享 / 公开页（Share）

> 用户把产物分享出去后，**别人**看到的那一页。Kokoro 增长玩法的核心触点。

---

## 一句话用途

让一个**没听说过 Kokoro 的人**，因为看到一个好东西，自己也想做一个。

---

## ASCII 布局速写

```
┌────────────────────────────────────────────────────────────────────┐
│              kokoro · 一只小图标 + "Kokoro" 字样（极小，左上）        │
│                                                                    │
│                                                                    │
│                    ┌──────────────────────────┐                    │
│                    │                          │                    │
│                    │                          │                    │
│                    │     产物本身             │                    │
│                    │     (海报 / 落地页 / 课件) │                    │
│                    │                          │                    │
│                    │                          │                    │
│                    └──────────────────────────┘                    │
│                                                                    │
│                   ── 来自 @用户名 · 用 Kokoro 做的 ──                │
│                                                                    │
│                                                                    │
│              ┌────────────────────────────────┐                    │
│              │      用 Kokoro 做一个 →          │   ← 主 CTA          │
│              └────────────────────────────────┘                    │
│                                                                    │
│              [💬 30 评论]   [🔁 复用 12]   [⭐ 转模板]                │
│                                                                    │
│              （评论区在下方展开 / 折叠）                              │
└────────────────────────────────────────────────────────────────────┘
```

- **产物居中放大**，占视觉重心
- 顶部 Kokoro 印记**极克制**，不抢戏
- 主 CTA "用 Kokoro 做一个"在产物正下方，醒目但不咄咄
- 评论 / 复用 / 转模板入口在 CTA 之后

## 关键组件

| 组件 | 说明 |
|---|---|
| 顶部 Kokoro 印记 | logo + 字样，10-14px，灰阶，左上 |
| 产物渲染区 | 居中放大，背景与 Kokoro 主区一致（极淡暖色） |
| 作者署名 | "来自 @用户名 · 用 Kokoro 做的"。点击进入作者主页（若开放） |
| 主 CTA | "用 Kokoro 做一个 →" 暖色填充 pill，深色文字 |
| 评论入口 | 💬 N 评论。点击展开评论区 |
| 复用按钮 | 🔁 复用 N。点击 = 把这个产物**复制到我的画布**继续改 |
| 转模板 | ⭐ 转为模板（仅产物作者可见） |
| OG 元数据 | `og:image` 用产物截图（带印记），`og:title` / `og:description` 用作者首句 |

## 关键状态

### A. 未登录访客（最关键）

- **整页可看可滚**，不弹登录墙
- 点 [复用] [评论] [转模板] 任意写操作 → 浮出**轻量登录提示**：
  - `想复用？登录一下，三秒搞定。`
  - 登录后直接落到目标动作（复用即开新 Canvas，评论即聚焦输入框）
- 点 CTA "用 Kokoro 做一个" → 跳到 [home.md](./home.md)（未登录态），保留来源 referer 用于归因

### B. 已登录访客

- CTA 不变，点击 = 进 Home 并自动 prompt："你看到的那个 XX 不错对吧？想做个什么样的？"（可选实验）
- 复用 / 评论直接进入流程，不再弹登录

### C. 作者本人

- 顶部出现一行小工具条：[编辑]（回 Canvas）/ [关闭分享]（链接失效）/ [改隐私]（仅自己 / 有链接可见 / 公开）
- "转模板"按钮高亮提示

### D. 链接失效 / 已私有

- 居中暖色插画 + 一行文案：`这个不再公开了。`
- 下方：`想做一个类似的？[去 Kokoro 看看]`

### E. 加载（首次进入）

- 产物区柔脉动占位，**不用 spinner**
- Kokoro 印记和 CTA 即刻可见，给"这是个 Kokoro 页"的判断

### F. Empty / First-time（产物刚分享出去，作者预览）

- 首次进入会顶部弹一条小提示（仅作者可见）：
  - `这是别人看到的样子。链接已复制到剪贴板。`
- 4s 自动消失

### G. 错误（产物渲染失败）

- 占位 + `这个产物加载出了点问题。[重试]`
- 不暴露技术细节

## 关键 copy 示例

### 作者署名

- `来自 @吴小心 · 用 Kokoro 做的`

### 主 CTA 候选

1. **`用 Kokoro 做一个 →`** —— 直接，默认
2. **`我也想做一个`** —— 第一人称，更软
3. **`换成你的版本 →`** —— 强复用感

> 倾向候选 1 作首发。候选 3 用于"复用"按钮的二次提示。

### 复用提示

- `想复用？登录一下，三秒搞定。`
- 复用后落到 Canvas："这是 @吴小心 的版本。你想改哪里？"

### 关闭分享

- `这个不再公开了。`

### 评论占位

- `留个想法？`（不用"评论一下"，更柔）

### 复制链接

- `链接已复制。`

## 增长设计要点（不可妥协）

1. **未登录全可看** —— 不能因为登录墙折损 80% 流量
2. **CTA 永远在视野内** —— 滚动时跟随或在产物结束处再出现一次
3. **OG 图必带印记** —— 朋友圈 / 微博 / Twitter 预览一眼可辨
4. **复用门槛极低** —— 一键复制到自己的 Canvas，不需要从头开始
5. **作者归因显眼** —— 鼓励作者继续分享，因为流量给到 ta
