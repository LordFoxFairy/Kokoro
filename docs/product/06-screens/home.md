# 首屏 / 新对话（Home）

> 用户进入 Kokoro 的第一个画面。气质即上下文：留白、温度、不催。

---

## 一句话用途

让用户在 3 秒内知道"我能在这里说出我想做的事"，并在第 1 句话里就开始往 Canvas 走。

## 关键组件

| 组件 | 说明 |
|---|---|
| Sidebar | 与 [ia.md](../04-architecture/ia.md) 一致，280px |
| 引导语 | 大字 36-44px，常规字重，不 bold |
| 输入框 | 胶囊型，高度 56-64px。左 [+]（附件/工具），右 [↑]（发送） |
| 产出物 chip × 5 | 见 [canvas-types.md](../03-product-form/canvas-types.md) |

| Sidebar 用户区 | 底部，未登录时显示"登录"按钮 |

## 关键状态

### A. 未登录（landing / 试用态）

- 引导语 + 输入框 + chip **都可见可点**
- 用户输入第一句话 / 点 chip 后：
  - 允许**生成预览**（前几轮免登录，到达存储 / 分享时再要登录）
  - 或：第一句话之前要求登录
  - → 待答题：登录门槛位置（见下）
- Sidebar 折叠，仅显示 logo + 登录按钮
- 顶部极小一行："Kokoro · 把想法做出来"

### B. 已登录新用户（首次进入）

- 引导语用更具引导性的变体（候选 2 / 3）
- chip 下方一行更小字提示："不知道做什么？点一个 chip 试试。"
- Sidebar 历史为空，显示"还没有对话"

### C. 已登录回访用户

- 引导语用默认（候选 1）
- chip 个别可被替换为"接着上次的 XX"（来自最近草稿）
- Sidebar 历史正常显示
- 可选：顶部一行"你 3 天前做的 XX 还在草稿里"软提示

### D. Loading（少见，仅首次刷新）

- 引导语 / 输入框骨架（柔脉动），不用 spinner

### E. Error（拉取失败、网络异常）

- 输入框 + chip 不变（即使断网也能本地起草）
- 顶部一行："网络有点慢，我先帮你记下来，连上再发。"

## 关键 copy 示例

### 引导语候选

1. **"今天想做什么？"** —— 默认、克制、不催
2. **"把想法说给我。"** —— 更主动、强调对话感
3. **"想做点什么？我陪你想。"** —— 强人格、回访友好

> 倾向候选 1 作首发，候选 2/3 用于新用户 / 回访做 A/B。

### 输入框 placeholder

- 默认：`把想法说给我……`
- 已选 chip 后：`告诉我更多 —— 海报想表达什么？`

### chip 标签（与 canvas-types.md 对齐）

- 🎨 小红书风海报
- 📊 一页落地页
- 📝 学习课件
- 💡 想法可视化
- ✨ 更多……

### 未登录提示（触达存储 / 分享时）

- "做到这一步想要存下来，登录一下吧。我替你把刚才的留着。"

### 网络异常

- "网络有点慢，我先帮你记下来，连上再发。"
